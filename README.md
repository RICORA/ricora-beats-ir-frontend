## Development

### Install

```sh
npm install
```

### Set environment variable

```sh
export SERVER_URL="https://example.com"
```

### Fetch beatmap repository

```sh
npx degit RICORA/ricora-beats-beatmap ./public/beatmaps
```

### Run development server

```sh
npm run dev
```

### Build

```sh
npm run build
```
